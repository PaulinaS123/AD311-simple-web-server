# Simple Node.js Web Server

AD 311 - Victoria Salomon

This project is a basic web server built using Node.js.
It serves an HTML home page, displays a text response for an about page, and handles invalid routes with a 404 error.

## Features

- Serves an HTML file on the home route (`/`)
- Displays a plain text message on the about route (`/about`)
- Handles unknown routes with a 404 response
- Built using Node.js core modules (`http` and `fs`)

## Technologies Used

- Node.js
- http module
- fs module
